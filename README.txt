CCE (Calculadora de Calificaciones Express) v.1.0 hecho por: Fofo Hernández usando Python

CCE es un programa utilizado para calcular tu promedio del año en base a los promedios obtenidos en cada uno de los tres trimestres que forman el año.

También se puede usar para saber si puedes o no exentar esa materia, así como para saber con qué promedio la puedes exentar. Esto es considerando 8.5 como el promedio con el que se exenta una materia.

En caso de no poderla exentar, puedes calcular el puntaje que necesitas en tu examen final para aprobar la materia. 

Por el momento la aplicación unicamente puede ser instalada y utilizada en computadoras y laptops.
